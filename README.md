# Subscription Expense Tracker

## Description
A streamlined dashboard designed to monitor and manage recurring subscription expenses. This tool helps users visualize their monthly and yearly financial commitments.

## Key Functionalities
* Automated tracking of recurring subscription costs.
* Data visualization for financial insights.
* Categorization of expenses to identify savings.

## Technical Stack
* JavaScript, React, Next.js

## Setup
1. Clone the repository: `git clone [repository-url]`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
